# rindexer Website

Marketing website for [rindexer](https://github.com/joshstevens19/rindexer) - a high-performance EVM indexing tool built with Rust.

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- Remotion (video)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
